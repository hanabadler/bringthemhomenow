We demand the safe return of all citizens who have been taken hostage by the terrorist group Hamas. We will not rest until every hostage is released and returns home safely. You can help bring them back home.
 
== Description ==
 
A WordPress plugin to add small snippet of code that place on the right bottom of the screen a curret timer of the number of days  citizens who have been taken hostage by the terrorist group Hama
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
Once Activated, the Timer will be displayed on the bottom right of the screen
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
